HOW TO RUN PROJECT IN ECLIPSE STS

1. Open Eclipse STS
2. File -> Import -> Existing Maven Project
3. Select this project folder
4. Wait for Maven dependencies download
5. Run PrimezatApplication.java
6. Open browser:
   http://localhost:8080

Pages:
Landing Page -> /
Login Page -> /login
Dashboard -> /dashboard

H2 Console:
http://localhost:8080/h2-console

JDBC URL:
jdbc:h2:mem:testdb
